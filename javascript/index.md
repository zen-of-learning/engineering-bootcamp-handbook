# 🟨 JavaScript Frontend Bootcamp

> **Goal:** In 2 intensive weeks, learn the essential JavaScript needed to build real frontend apps, handle user input, connect to APIs, and be ready to start React.

Each lesson is designed for **30–45 minutes** of focused work. Read the short explanation, type the examples, complete the exercises, then finish the mini-task or project step.

---

## 🎯 By the End You Can

- Write clean JavaScript using ES6+ basics
- Work confidently with arrays and objects
- Manipulate the DOM to update webpages
- Handle clicks, input changes, and form submissions
- Use `fetch`, `async`, `await`, and `try/catch`
- Display API data in the browser
- Build small frontend apps with separate HTML, CSS, and JavaScript files
- Understand enough frontend JavaScript to start React next

---

## 🔥 Learning Rules

- Keep theory short and practical
- Code every day
- Use real-world examples: users, products, forms, todos, and API data
- Prefer small working apps over abstract exercises
- Skip deep closures, prototypes, advanced `this`, and full React for now
- Finish each day with something you can run in the browser
