# Day 13–14 — Final Project: Startup Landing Page + Dynamic Section

> **Goal:** Build a small real frontend project using static sections, user interaction, form input, and API data.

---

## Short Explanation

The final project combines the skills from the last 12 days. You will build a startup landing page with normal static content and one dynamic API-powered section. The static sections show that you can structure a page. The dynamic section shows that you can fetch data and render it. The interactions show that you can handle clicks and forms. Keep the project small but complete. Use separate files, clear function names, and a simple design. This is not a React project yet, but the habits you practice here prepare you for React components, state, and API-driven UIs.

---

## Project Requirements

Build **Startup Landing Page + Dynamic Section**.

Static sections:

- Who we are
- Sponsors

Dynamic section:

Add the button interaction:

```js
const learnMoreButton = document.querySelector("#learn-more-button");
const comingSoonMessage = document.querySelector("#coming-soon-message");

learnMoreButton.addEventListener("click", () => {
  comingSoonMessage.textContent = "Beta access opens soon. Stay tuned!";
});
```

## Day 14 — Add API Data, Form Handling, and Polish

Fetch posts and render them as updates:

```js
const statusMessage = document.querySelector("#status");
const updatesContainer = document.querySelector("#updates");

function renderUpdates(posts) {
  updatesContainer.innerHTML = "";

  posts.slice(0, 3).forEach((post) => {
    const article = document.createElement("article");
    article.classList.add("update-card");
    article.innerHTML = `
      <h3>${post.title}</h3>
      <p>${post.body}</p>
    `;
    updatesContainer.appendChild(article);
  });
}

async function loadUpdates() {
  try {
    statusMessage.textContent = "Loading updates...";

    const response = await fetch("https://jsonplaceholder.typicode.com/posts");

    if (!response.ok) {
      throw new Error("Failed to load updates");
    }

    const posts = await response.json();
    statusMessage.textContent = "";
    renderUpdates(posts);
  } catch (error) {
    statusMessage.textContent = "Could not load updates right now.";
  }
}

loadUpdates();
```

Handle the waitlist form:

```js
const signupForm = document.querySelector("#signup-form");
const emailInput = document.querySelector("#email-input");
const formMessage = document.querySelector("#form-message");

signupForm.addEventListener("submit", (event) => {
  event.preventDefault();

  const email = emailInput.value;

  if (email === "") {
    formMessage.textContent = "Please enter your email.";
    return;
  }

  formMessage.textContent = `Thanks! We will send updates to ${email}.`;
  signupForm.reset();
});
```
